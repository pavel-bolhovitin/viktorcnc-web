# viktorcnc-web

## Environment Setup

1. NVM (<https://github.com/nvm-sh/nvm#installing-and-updating>)

   Node.js 22.22.0 (via nvm)

   ```bash
   nvm install 22.22.0
   nvm use 22.22.0
   ```

   Corepack (for pnpm)

   ```bash
   corepack enable
   ```

   pnpm 10.30.3

   ```bash
   corepack prepare pnpm@10.30.3 --activate
   ```
